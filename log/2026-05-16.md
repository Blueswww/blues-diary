### 10. 昵称输入框真机不显示已输入文字（核心 Bug）
- **现象**：模拟器正常，真机调试输入框不显示当前的昵称文字
- **排查过程**：
  1. 添加 `type="text"` → 无效
  2. `v-if` 改 `v-show`（`hidden` 属性）→ 无效
  3. 用 `nextTick` 分两次渲染设值 → 无效
  4. CSS `visibility: hidden; height: 0` 折叠替代隐藏 → 无效
- **根因**：微信小程序真机上 `<input>` 是**原生组件**，通过 `wx:if`/`hidden`/CSS 折叠从不可见→可见时，原生组件会重新初始化但读不到 `value` 绑定的初始值
- **最终方案**：**input 始终渲染、永不隐藏**。用 `readonly` + CSS 样式切换代替显示/编辑模式切换
  - 显示态：`readonly`、无边框透明背景、居中粗体，看起来就是昵称文字
  - 编辑态：非 `readonly`、有边框灰背景、左侧标准输入框
  - 只有按钮用 `v-if` 条件渲染（非原生组件无此问题）
- **涉及文件**：`src/pages/mine/mine.vue`（模板、样式、逻辑全面重写）
