# 开发日志 2026-05-16

## 已完成功能

### 1. 节假日系统重构 — 算法自动计算
- **目标**：去掉手动维护的节假日数据，改为算法每年自动计算
- **方案**：使用 `lunar-javascript` 库进行农历→公历转换
- **实现**：
  - 新增 `src/utils/holidayCalculator.ts` — 根据年份算法化计算所有节日
  - 重写 `src/store/holiday.ts` — 去掉云端依赖，本地计算 + localStorage 缓存
  - 每年第一次打开自动计算当年 + 下一年数据，无需任何手动更新
- **涵盖 30+ 个节日**：
  - 农历节日：春节、元宵、端午、七夕、中元、中秋、重阳、腊八、小年、除夕（通过 `lunar-javascript` 转换）
  - 固定公历节日：元旦、情人节、妇女节、植树节、愚人节、劳动节、青年节、儿童节、建党节、建军节、教师节、国庆节、万圣节、平安夜、圣诞节、跨年夜
  - 浮动节日：母亲节（5月第2个周日）、父亲节（6月第3个周日）、感恩节（11月第4个周四）
  - 节气节日：清明节（通过 `getJieQi()` 检测）
- **清理**：删除了 `cloudfunctions/holidays/` 云函数、`src/api/holiday.ts` API 模块、`cloudbaserc.json` 中的 holidays 条目
